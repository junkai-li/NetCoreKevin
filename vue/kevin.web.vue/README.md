# AI智能体后台管理系统 - 使用文档

## 一、系统概述

### 1.1 系统简介

**AI智能体后台管理系统**是一款基于 Vue 3 + Ant Design Vue 构建的企业级后台管理平台，主要用于管理AI智能体、用户、权限、知识库等核心业务模块。

### 1.2 技术架构

| 层级 | 技术栈 | 说明 |
|------|--------|------|
| 前端框架 | Vue 3 | 渐进式 JavaScript 框架 |
| UI组件库 | Ant Design Vue 4.x | 企业级UI设计语言 |
| 路由管理 | Vue Router 4.x | Vue官方路由管理器 |
| HTTP客户端 | Axios | 基于Promise的HTTP库 |
| 实时通信 | SignalR | 实时双向通信 |
| 代码编辑器 | CodeMirror 6 | 高性能代码编辑器 |
| 构建工具 | Vue CLI 5 | Vue官方构建工具 |

### 1.3 系统环境要求

- **Node.js**: >= 14.0.0
- **npm**: >= 6.0.0
- **浏览器支持**: Chrome >= 80, Firefox >= 75, Safari >= 13, Edge >= 80

## 四、核心功能详解

### 4.1 文件上传

在AI聊天页面支持文件上传功能：

1. **上传操作**:
   - 点击上传按钮
   - 选择一个或多个文件
   - 文件将显示在上传列表中

2. **发送逻辑**:
   - 发送消息时自动携带文件信息
   - 文件名为 `FileNames` 字段（逗号分隔）
   - 文件URL为 `ContentFileUrls` 字段（逗号分隔）
   - 发送成功后自动清空上传列表

3. **文件展示**:
   - 消息下方展示文件链接
   - 支持点击下载

### 4.2 链接高亮

消息内容中的URL会自动高亮并可点击：

- 支持 `http://` 和 `https://` 协议
- 自动识别Markdown格式中的链接
- 点击后在新窗口打开

### 4.3 消息复制

所有消息内容（包括用户发送和AI回复）都支持复制：

- 点击消息旁的复制按钮
- 支持现代浏览器剪贴板API
- 旧浏览器自动降级处理

### 4.4 智能体安全拦截

智能体支持安全拦截功能：

- **开启状态**: 对Python脚本和Shell命令内容进行安全拦截
- **默认状态**: 开启
- **配置位置**: 智能体编辑页面的 `IsSecurityIntercept` 开关

## 六、登录与权限

### 6.1 登录流程

1. 访问系统首页自动跳转至登录页
2. 输入账号密码进行登录
3. 登录成功后跳转至仪表盘

### 6.2 权限验证

- 系统采用路由守卫进行权限验证
- 未登录用户自动重定向至登录页
- 用户信息缓存有效期为10分钟

## 八、常见问题

### 8.1 页面加载异常

**问题**: 页面空白或加载缓慢

**解决方案**:
1. 检查网络连接
2. 清除浏览器缓存
3. 检查控制台错误信息
4. 联系系统管理员

### 8.2 登录失败

**问题**: 账号密码正确但无法登录

**解决方案**:
1. 检查账号密码是否正确
2. 检查账号是否被禁用
3. 联系管理员重置密码

### 8.3 文件上传失败

**问题**: 文件上传提示失败

**解决方案**:
1. 检查文件大小（建议不超过10MB）
2. 检查文件格式是否支持
3. 检查网络连接

### 8.4 分页功能异常

**问题**: 分页显示异常或数据重复

**解决方案**:
1. 刷新页面重新加载
2. 检查分页参数配置
3. 联系开发人员排查接口问题
